## Tools

This project requires **Node.js 22 (LTS)**. We recommend using [nvm](https://github.com/nvm-sh/nvm) to manage your Node versions. A `.nvmrc` file is included in the root.

To use the correct version:
```bash
nvm use
```

- [Tailwind CSS](https://tailwindcss.com/)
